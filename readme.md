# Layoffs Data Cleaning Project (MySQL)

# Project Overview
Performed data cleaning on layoffs dataset using MySQL.

# Tools Used
  MySQL
  SQL
  CSV Dataset

# Data Cleaning Steps
  Removed duplicates
  Handled NULL values
  Standardized text fields
  Converted date formats
  Removed unnecessary columns

# Skills Demonstrated
  DELETE
  UPDATE
  CASE statements
  Window functions
  Data transformation

# How to Run
1. Import raw_data.csv into MySQL
2. Run data_cleaning_queries.sql

# Exploratory Data Analysis (EDA) – Layoffs Dataset

This project performs Exploratory Data Analysis on a cleaned layoffs dataset using SQL. The analysis identifies trends, top affected companies and industries, country-wise layoffs, and year-wise patterns using aggregation and window functions.
